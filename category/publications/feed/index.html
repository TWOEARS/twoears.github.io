It allows for numerical simulations of sound fields in the time and frequency domain. Furthermore, with the generation of binaural simulations of multichannel loudspeaker setups it allows for creating stimuli for listening tests in order to evaluate different sound field synthesis methods.

After a few years of work, today we finally are coming up with the 1.0.0 release.
You can download the latest release and you should have a look at the tutorial on github how to use it.
